
## المميزات

- تسجيل حساب مع إرسال كود تحقق للبريد الإلكتروني
- محادثات ذكية متعددة مع نموذج Llama 3.3 70B
- حفظ تلقائي لكل المحادثات في المتصفح
- تصدير المحادثات كملف JSON
- استيراد محادثات من ملف JSON
- واجهة عربية بالكامل (RTL)
- تصميم متجاوب يعمل على كل الأجهزة
- خلفية جسيمات تفاعلية مع الماوس

## الإعداد

1. افتح `js/auth.js` وغيّر `GROQ_API_KEY` بمفتاحك من console.groq.com
2. غيّر `VERIFY_EMAIL` بإيميلك (تأكد من تفعيله عبر formsubmit.co)
3. افتح `index.html` في المتصفح

## النشر على GitHub Pages

1. أنشئ مستودع جديد على GitHub باسم `rtq-ai`
2. ارفع كل الملفات بنفس الهيكل
3. اذهب إلى Settings → Pages → Branch: main → Save
4. انتظر دقيقتين ثم افتح `https://اسمك.github.io/rtq-ai/`
